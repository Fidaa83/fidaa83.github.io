# Personal Portfolio Website

A personal portfolio website built using **HTML and CSS** to present a professional profile, selected projects, and technical skills in a clean and structured way.

---

## Overview
This project is a static portfolio website designed to give a clear and focused overview of my background as a software engineer.  
The layout is simple, readable, and responsive across different screen sizes.

---

## What This Project Does
- Displays a short professional summary (About section)
- Showcases selected projects with descriptions
- Presents technical skills grouped by category
- Provides quick access icons for documents and profiles
- Maintains a clean and consistent visual structure

---

## Features
- Responsive layout using modern CSS
- Clear section separation (About, Projects, Skills)
- Card-based project presentation
- Organized and readable typography
- Icon-based quick access to documents and profiles
- Easy to maintain and extend

---

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- Basic JavaScript (for small dynamic elements)

---

## Purpose
The purpose of this project is to serve as a personal portfolio website that clearly presents my skills and projects, and can be easily updated as I gain more experience.
